# MensCharmCrumblesImStillHere

> 全世界男性魅力下降一百倍而我不变

![](./images.jpg)

除主角以外，世界上所有男性角色的魅力值按比例缩放至选定等级区间内。

~~防止玩家被牛。~~

## 模组配置

可通过游戏内 Mod 设置面板调整「最高魅力」等级。采用**比例缩放**而非硬截断：

```
新魅力 = floor(原魅力 × 区间上限 / 900)
```

当原魅力 = 900 时，结果为 `区间上限 - 1`，确保不跨越到上一级。

| 选项 | 魅力区间 | 缩放范围 |
|------|----------|----------|
| 非人 | [0, 100) | [0, 99] |
| 可憎 | [100, 200) | [0, 199] |
| 不扬 | [200, 300) | [0, 299] |
| 寻常 | [300, 400) | [0, 399] |
| 出众 | [400, 500) | [0, 499] |
| 瑾瑜 | [500, 600) | [0, 599] |
| 龙姿 | [600, 700) | [0, 699] |
| 绝世 | [700, 800) | [0, 799] |
| 天人（无效果） | [800, 900] | 不修改 |

默认选项为「天人」（不做任何修改），需手动选择等级生效。

## 实现

后端 Harmony Postfix 补丁，钩住 `Character.GetAttraction()`：
- 非太吾 + 男性 → 按 `MaxCharm` 配置进行比例缩放
  - `MaxCharm = 1~8`：`result = floor(result × 区间上限 / 900)`，900 特殊映射为 `区间上限 - 1`
  - `MaxCharm = 9`（天人/默认）：不修改

## 目录结构

## 构建

```bash
cd src
dotnet build
```

DLL 自动输出到 `../Plugins/`。

## 安装

将整个文件夹放入：
```
The Scroll Of Taiwu/Mod/
```
